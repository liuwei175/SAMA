# Breast Cancer Robust Logistic Experiment

This folder contains the final Breast Cancer experiment package used to
reproduce the selected result tables.

## Contents

- `data/raw/breast_cancer.csv`: Breast Cancer input data generated from `sklearn.datasets.load_breast_cancer`.
- `config/final_selected_configs.csv`: fixed model configurations used in the final tables.
- `scripts/train_final_models.py`: data processing, sample-level splitting, Direct/Majorant/ERM training, and exact robust evaluation.
- `scripts/build_final_tables.py`: aggregation, advantage tables, and Direct/Majorant fairness audit.
- `data/processed/final_split_results.csv`: final split-level results generated by the training protocol.
- `results/`: generated summary tables and audit files.

## Training Protocol

Dataset:

- Wisconsin Breast Cancer data.
- Sample-level stratified train/validation/test splits.
- Split seeds: `202600, 202601, 202602, 202603, 202604`.
- Features are standardized using the training subset of each split.

Uncertainty model:

- Five feature groups: size, texture, surface, irregularity, shape.
- `epsilon in {0.15, 0.25, 0.35}`.
- `N in {1, 2, 4, 8, 16}`.
- Uniform nested uncertainty bank: the same bank prefix is used for Direct and Majorant at each `(split, epsilon, N)`.
- Exact test and validation evaluation use all box corners.

Model training:

- Direct and Majorant use the same split, uncertainty bank, bank prefix, initialization, smoothing schedule, projection box, and number of updates.
- Initialization: `w=0, b=0`.
- Projection: `w in [-2, 2]`.
- Stage updates: `600` per `N`, total `3000`.
- ERM is trained on unperturbed data with the clean logistic objective and then evaluated using the same exact robust/CVaR metrics.

Final configurations are listed in `config/final_selected_configs.csv`.

## Reproduce

Install dependencies:

```bash
pip install -r requirements.txt
```

Regenerate split-level training results:

```bash
python scripts/train_final_models.py
```

Regenerate final tables:

```bash
python scripts/build_final_tables.py
```

Primary outputs:

- `results/final_loss_table.csv`
- `results/advantage_majorant_vs_erm_reference.csv`
- `results/advantage_majorant_vs_direct.csv`
- `results/fairness_audit_direct_majorant.csv`
- `results/tables.md`
